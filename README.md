
# 1C Matrix — Telegram Mini App

Современная система управления задачами и командой в стиле СБИС / 1С, работающая прямо внутри Telegram.

## Основные функции
- **Ролевая модель**: Администратор (создание задач/команды) и Исполнитель (вход по коду).
- **Изоляция данных**: У каждого пользователя своя база данных (привязана к Telegram ID).
- **Сохранение данных**: Автономная работа базы данных на сервере.
- **Дизайн**: Современный Glassmorphism интерфейс.

## Технологический стек
- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express.
- **Deployment**: Render.com.

## Как запустить локально
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите клиентскую часть:
   ```bash
   npm run dev
   ```
3. Запустите сервер:
   ```bash
   npm run server
   ```

## Деплой на Render.com
1. Создайте **Web Service**.
2. Подключите этот репозиторий.
3. Команда сборки: `npm install && npm run build`
4. Команда запуска: `node server.js`
5. Добавьте диск (Disk) в настройках Render для сохранения файла `database.json`.

## Интеграция с Telegram
Создайте бота через @BotFather и привяжите Mini App к URL вашего сервера на Render.
